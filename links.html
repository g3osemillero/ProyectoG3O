Software

GAMMIT/GLOBK

Dentro del semillero se cuenta con la licencia para el software de procesamiento GAMMIT/GLOBK, el cual permite procesar datos GNSS, VLBI y experimentos SLR. GAMMIT/GLOBK fue desarrollado en Massachusetts Institute of Technology (MIT) bajo la dirección del Dr. Robert W. King. Para solicitar la licencia de este software se debe hablar directamente con el profesor Andrés Cárdenas Contreras.

Web Page Gammit/Globk

GPSTk

GPS Toolkit es un software desarrollado por la Universidad de Texas. Su ideal era ser una libreria de C++ abierta a todo el público para el procesamiento de datos GNSS, sin embargo como se menciona en su repositorio oficial la libreria actualmente se encuenta en desuso, por tal motivo se lista el links a las librerias que recomienda el autor para su uso.

GNSSTk (Libreria) GNSSTk (Aplicaciones)

gLAB

gLAB es un software desarrollado por el grupo de Astronomia y Geomatica (gAGE) de la Universidad Politecnica de Cataluña (UPC) para la Agencia Espacial Europea (ESA), este software permite realizar modelos precisos de Observaciones GNSS.
